# bigScreenExample
主要展示一种监控大屏的浏览器兼容性方案，示例可以适应任何分辨率，并无滚动条。
##无滚动条的浏览器兼容性方案
思路主要分为2步：

**1 将UI设计稿等比例缩放到屏幕上。**

**2 使用flex将设计稿横向或纵向拉伸以覆盖整个屏幕。**

`由于UI设计稿的宽高比，与屏幕宽高比不等，造成浏览器会出现一片空地（空白区域）；
